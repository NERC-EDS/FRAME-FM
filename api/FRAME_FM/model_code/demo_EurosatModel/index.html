FRAME_FM.model_code.demo_EurosatModel

Example Model code for processing EuroSAT dataset

Two models are available. 1. a Pretrained ResNet18 with weights from PyTorch Model Zoo 2. An autoencoder from a [Nature Paper on Flood Analysis](https://www.nature.com/articles/s41598-025-96781-2)

This model consists of a convolutional encoder that progressively increases feature channels through stacked Conv-ReLU blocks,

followed by a latent projection layer that compresses the final feature map into a fixed-size vector via adaptive average pooling and a linear layer. A corresponding inverse mapping (self.from_latent) expands latent vectors back into channel-first feature tensors, supporting downstream decoding or reconstruction.

> Thappitla, R.S., Villuri, V.G.K. & Kumar, S. An autoencoder driven deep learning geospatial approach to flood vulnerability analysis

in the upper and middle basin of river Damodar. Sci Rep 15, 33741 (2025). https://doi.org/10.1038/s41598-025-96781-2
